ASL to Text

Image Captioning

Describes the image provided by the user into text caption. Helps people with visual impairment to by generating a textual description of the image.

Speech-to-Text

how it works

1

ASL-to-Text

Live captured images are converted into vectors using pre-trained MobileNet CNN model. This trained dataset is used to predict the new gesture which is done using kNN classification.

2

Image Caption

Images can be provided by users and can be interpreted by using both Convolutional and recurrent neural networks (Resnet50 architecture and LSTM) to give apt captions for the same.

3

Speech-to-Text

The user speaks through a microphone, the device detects the voice and converts it to text using open-source Web Speech API to convert speech to text in real-time.

frequently asked questions

Do I require a special kind of gear/glove for gesture detection?

No, there is no requirement of any additional equipment. You will only require a decent PC/laptop with web camera.

Is there any specification for image resolution?

No, there are no specifications on image resolution or dimension, just upload the image in jpg or jpeg or png format.

Why speech-to-text feature is not working in my browser?

Speech-to-Text feature uses Web-Speech-API which is not compatible with all browsers. We recommend you to use Google Chrome. For more information, please visit this website.

Do I need to train the model every time I want to use it?

No, you will need to train your model only when you are using it for the first time. Once created, you can download your dataset. The next time you want to use the same dataset, you can simply upload that .json file and you are good to go!
